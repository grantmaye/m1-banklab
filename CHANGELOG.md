# Changelog

## 0.1.0 - 2026-07-08

- Initial Spring Boot 3 modular monolith scaffold.
- Added PostgreSQL schema with Flyway migrations and seed data.
- Added customer and account APIs.
- Added deposit, withdrawal, and transfer flows.
- Added audit logging for financial actions.
- Added MVP fraud scoring rules and fraud assessment API.
- Added Docker Compose, OpenAPI/Swagger, JUnit 5, Testcontainers, and GitHub Actions CI.
